# Dossier Reports

Ce dossier contient tous les rapports de tests générés par les différents outils.

## Structure

- `postman-report.html` : Rapport HTML des tests Postman
- `postman-results.xml` : Résultats JUnit des tests Postman
- `jmeter-report/` : Dashboard HTML des tests JMeter
- `jmeter-results.jtl` : Fichier de résultats bruts JMeter
- `chaos-http.json` : Journal des tests Chaos HTTP
- `chaos-cpu.json` : Journal des tests Chaos CPU
- `chaos-auth.json` : Journal des tests Chaos authentification
- `index.html` : Page d'accueil consolidée des rapports (généré par le pipeline)

## Note

Les rapports Selenium sont disponibles dans `selenium/target/surefire-reports/`
